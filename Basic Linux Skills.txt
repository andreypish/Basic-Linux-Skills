У каждой команды есть много дополнительных опций. Чтобы узнать подробнее о команде нужно ввести "<команда> --help
Если команда выполняется слишком долго, ее можно завершить, нажав Ctrl+C
КОМАНДЫ Linux:
date - показать текущую дату и время системы.
pwd - посмотреть текущую директорию(где я?)
ls - показать содержимое в текущей директории 
file - посмотреть тип файла
cd - пройти в другую директорию
echo > - создать файл
touch - создать несколько файлов
mkdir - создать директорию
echo - вывод строки на экран/запись текста в файлы
cat - посмотреть содержимое файла
rm - удалить файл/директорию
cp - копирование файлов/каталогов
mv - перемещение/переименование 
less - посмотреть содержимое большого файла
head - показывает несколько первых строк из файла
tail - показывает несколько последних строк из файла
find - поиск файлов и каталогов
grep - поиск строки или слов в файлах
vim - открыть текстовый редактор


Создание sh-скриптов позволяет легко объединять множество команд.
Пример простого скрипта:
#!/bin/bash
 
mkdir d1
cd d1
touch f1.txt f2.txt f3.txt
ls -al
echo Hello > f1.txt
cat f1.txt
